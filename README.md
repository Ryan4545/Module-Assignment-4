# Module-Assignment-4
Clustering DC-Area Crash Hotspots: What the Road Network Tells Us

Make sure to download the datasets that this takes advantage of:
https://opendata.dc.gov/datasets/roadway-block/explore
https://opendata.dc.gov/datasets/crashes-in-dc/explore

Download the CSV files and the Shapefile zip folder and extract its contents to the root. Once this is done for both datasets, make a virtual environment, install all pip dependencies, and run full.py to get the figures and data analysis generation.
